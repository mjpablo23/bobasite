Analysis of Metallic Elements



Development of plasma in solution for various applications. One promising example is metallic element analysis using plasma optical emission spectroscopy.

Reference: C.-Y. Wang and C.-C. Hsu, Environmental Science & Technology 53 (18), 10888-10896 (2019).


Machine Learning



Machine learning using plasma spectroscopy for analysis purposes. Following shows an example using CNN for discriminant of volatile organic compounds.

Reference: C.-Y. Wang, T.-S. Ko and C.-C. Hsu, Analytica Chimica Acta 1179, 338822 (2021).


Portable Plasma System



Development of various portable and low cost plasma systems. Following shows a portable plasma power source that can be modulated using a mobile phone with a Bluetooth-module.

Reference: NTUChE Plasma Engineering Lab.


Materials Processing




Using various plasmas sources for materials processing. One novel example is using a microplasma generation device to fabricate microfluidic paper-based analytical devices.

Reference: P.-K. Kao and C.-C. Hsu, Anal. Chem. 86, 8757 (2014).
